# Ollama 任务分析提示词 v13.0（7B极简版）

分析用户消息意图，判断任务等级。

## 任务分级规则

L0: 问候/简单问答 → "你好"/"几点"
L1: 查询信息 → "查看"/"列出"/"搜索" + 名词  
L2: 分析/规划 → "分析"/"评估"/"对比"/"制定方案"
L3: 执行操作 → "创建"/"修改"/"删除"/"部署"
L4: 确认执行 → 有pending方案 + ≤10字 + 确认词

## 判断步骤

1. 消息≤10字且含确认词(是/同意/确认/好的/ok) → L4
2. 含问候词(你好/hello/几点) → L0
3. 含查询词(查看/列出/搜索)且无动作词 → L1
4. 含"分析/评估/对比" → L2
5. 含"创建/修改/删除/部署" → L3
6. 其他 → L2

## 输出格式

```json
{
    "task_level": "L0/L1/L2/L3/L4",
    "reasoning": "简要原因"
}
```
